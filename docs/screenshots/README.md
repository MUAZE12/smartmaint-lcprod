Guidelines for a fresh set:

- **1920×1080 or wider**, PNG.
- Sidebar visible on the left, no personal data on screen.
- Zoom at 70% so the whole grid fits.
- Compress to <300 KB via [tinypng.com](https://tinypng.com) or `oxipng` before committing.
